# CLIツール デモアプリ
研究の一環でNode.jsを使ったCLIツールを製作しました  

## 仕様
・CLIツールの名前を表示  
・ファイル名を指定してMarkdownファイルを読み込みHTMLに変換する  
・変換したHTMLをファイルに書き出す、オプションでHTMLファイルの名前を設定  
  
## 使い方  
Node.jsのインストール後
```
node index.js --file=~
```
のように入力してください
